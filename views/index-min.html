Price Calc


Price Calc is a stand alone cross platform application built for automatic
calculation of pricing for different channels and different brands.


Features


Options Description
Applies to PRICING, SHIPPING, EXCHANGE AND TAX RATES, COST CONSTRAINTS
View
  • Go to View tab in Pricing tab
  • Click on 'REFRESH' Button
  • The drop down box will be populated with the Channels
  • Select a channel of your interest from the drop box
  • Click 'SEARCH' Button
  • The Search results will be appear below within a table.
Add / Update
  • Go to 'Add / Update' tab in Pricing tab
  • To Add ,fill the input boxes and then, click 'ADD' Button.
  • If successful, a success message will be shown below
  • To Update, fill the input boxes, click 'UPDATE' Button
  • If successful, a success message will be shown below
Delete
  • Go to Delete tab of Pricing
  • Enter the channel and brand name in the input boxes
  • Click Delete and the enter will be deleted
                       Note ** : The File format should be Category, Model, Brand, CostUSD
